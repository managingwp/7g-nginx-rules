# AGENTS.md

## Description
* This repository contains custom rules for GridPane's nginx variant of the 7g firewall. These rules are designed to enhance the security of web applications by blocking or allowing specific requests based on certain criteria.
* Documentation on how to use the nginx 7G firewall on GridPane is at https://gridpane.com/kb/using-the-7g-web-application-firewall/
* Documentation on how to use the OLS 7G firewall on GridPane is at https://gridpane.com/kb/the-gridpane-7g-web-application-firewall-and-openlitespeed-ols/
